# Traductor de Lenguaje Natural a Emojis

Proyecto de análisis léxico y sintáctico usando ANTLR

Este proyecto utiliza ANTLR para procesar expresiones que relacionan frases de lenguaje natural con emojis, implementando análisis léxico y sintáctico.
